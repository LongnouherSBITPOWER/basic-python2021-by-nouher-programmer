ພາສາ HTML

HTML ຫຍໍ້າມາຈາກ (Hyper Text Markup Language) ເປັນພາສາທີ່ໃຊ້ໃນການພັດທະນາ Web page ເພຶອໃຫ້ໂປຣເເຣມ web brower ຕ່າງໆເຊັ່ນ:(Internet Explorer,Netscape ເເລະ Opera) ສາມາດເເປງຄຳສັ່ງ ສະເເດງຜົນໃນລັກສະນະຂອງຮູບ,ຕົວໜັງສື, ສຽງ ເເລະ ພາບເຄື່ອນໄຫວໂດຍໄຟລ໌ນາມສະກຸນ.html ຫຼື.htm.

90%


ພາສາ CSS

CSS ຫຍໍ້າມາຈາກ (Cascading Style Sheets) ເປັນພາສາທີ່ໃຊ້ສຳຫຼັບຕົກເເຕ່ງເອກະສານ HTML/XHTML ໃຫ້ມີໜ້າຕາສີສັນລະຫວ່າງຟື້ນຫຼັງ ເສັ້ນຂອບ ເເລະ ອື່ນໆຕາມທີ່ຕ້ອງການ.

70%


ພາສາ PHP

PHP ຫຍໍ້າມາຈາກ (Hypertext Preprocessor) ເປັນພາສາປະເພດ Script Language ທີ່ທຳງານເເບບ Server Side Script ຂະບວນການທຳງານເເບບໂປຣເເກຣມເເປຄຳສັ່ງ interpreter ຄືເເປພາສາທຸກຄັ້ງທີ່ມີຄົນເອີ້ນ Script.

40%


ພາສາ JavaScript

JavaScript ເປັນພາສາ script ວັດຖຸເຊິ່ງໃນການສ້າງ ເເລະ ພັດທະນາເວັບໄຊຮ່ວມກັບ HTML ເພືອໃຫ້ເວັບເບິ່ງມີການເຄື່ອນໄຫວສາມາດຕອບສະໜອງຜູ້ໃຊ້ງານໄດ້ເປັນຈຳນວນຫຼວງຫຼາຍ.

35%


ພາສາ C++

ພາສາ C++ ເປັນພາສາທີ່ຖືກອອກເເບບສຳຫຼັບການທຳງານພາຍໃຕ້ສິ່ງເເວດລ້ອມລະບົບປະຕິບັດການ UNIX ດ້ອຍພາສາ C++ ຜູ້ຂຽນໂປຣເເກຣມສາມາດຂຽນໂປຣເເກຣມໄດ້ຢ່າງມີປະສິທິພາບຫຼາຍຂຶ້ນ ນອກຈາກນີ້ການຂຽນໂປຣເເກຣມເພື່ອໃຫ້ສາມາດນຳກັບມາໃຊ້ໄດ້ໃໝ່(reusability)ກໍ່ສາມາດທຳໄດ້ງ່າຍຂື້ນ.

20%
